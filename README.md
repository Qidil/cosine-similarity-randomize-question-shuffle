# Penerapan Algoritma Cosine Similarity dalam Efektifitas Pengacakan Soal UJian Online 

Proyek tugas akhir untuk kelulusan yang menggunakan Algoritma Cosine Similarity, untuk mencegah kesamaan dan kemiripan soal

Untuk dokumentasi dari repository ini: https://journal.uin-alauddin.ac.id/index.php/instek/article/view/56473

---

## 🚀 Fitur Utama
- Sistem pengacakan menggunakan Algoritma Cosine Similarity dengan bantuan logika TF-IDF
- Pengoprasian data dengan python
- Validasi hasil denghan MAE dan RMSE

---

## 🛠️ Tech Stack
- Frontend: -
- Backend: Python menggunakan Cosine Similarity Algorithm, TF-IDF, MAE dan RMSE 
- Database: MySQL
